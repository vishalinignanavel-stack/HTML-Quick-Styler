# HTML Quick Styler

## Project Description
HTML Quick Styler is a dynamic web page generator built using Python and Gradio.  
It allows users to generate responsive HTML pages with customizable color themes.

## Features
- Dynamic Page Title
- Custom Description Input
- Multiple Color Themes
- Instant HTML Generation
- Public Deployment via Gradio

## Technologies Used
- Python
- Gradio
- HTML5
- CSS3
- Google Colab

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Click Generate Page
4. Access public demo link
